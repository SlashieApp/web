# Motion

The SDL motion system. Every duration and easing curve below is **read live from
`sdlMotion`** in `src/theme/styles.ts`, so this page can never drift from the code that
ships. Specimens render with SDL semantic color roles (`action.primary`, `bg.surface`,
`border.default`, …) defined in `src/theme/chakraSystem.ts`, so they follow the active
light/dark theme.

**Two non-negotiable rules govern all SDL motion:**

- **Animate `transform` and `opacity` only.** These are GPU-compositable and never trigger
  layout or paint. Never animate `width`, `height`, `top`, `left`, `margin`, or `box-shadow`
  geometry.
- **Honor `prefers-reduced-motion`.** Users who request reduced motion must get an instant,
  non-animated result. The live demo below wraps its transition in
  `@media (prefers-reduced-motion: reduce)` and collapses the duration to `0ms`.

Per the green-ink rule, the green fills on this page pair with `text.onGreen` ink, never
white.

## Durations

Reach for a named duration token rather than a hardcoded `ms` value. Shorter tokens are for
small, frequent UI feedback; longer tokens are reserved for larger spatial transitions and
the map.

export const DURATION_USE = {
  instant: 'Press/active state feedback; checkbox ticks.',
  fast: 'Hover color shifts, small icon swaps.',
  base: 'Default control transitions (buttons, inputs).',
  moderate: 'Border/shadow transitions, form focus rings.',
  slow: 'Drawers, modals, larger entrances/exits.',
  map: 'Map pans, zooms and camera moves only.',
}

## Easings

SDL uses four easing curves. `standard` is the default for nearly everything; `emphasis`
overshoots slightly and is reserved for confirmation moments (quote accepted, job closed).

## Live demo

A `transform` + `opacity` transition built entirely from `sdlMotion` tokens. It uses
`sdlMotion.duration.moderate` and `sdlMotion.easing.standard`, read live from the theme.
The transition is wrapped in a `prefers-reduced-motion` media query: if you have reduced
motion enabled, the chip snaps to its new position instantly with no animation.

export const MotionDemo = () => {
  const [on, setOn] = useState(false)
  const duration = sdlMotion.duration.moderate
  const easing = sdlMotion.easing.standard
  return (
    <Stack gap={4} maxW="520px">
      <Box
        position="relative"
        h="96px"
        borderRadius="lg"
        bg="bg.surface"
        borderWidth="1px"
        borderColor="border.default"
        boxShadow="e1"
        overflow="hidden"
      >
        <style>{`
          .sdl-motion-chip {
            transition: transform ${duration} ${easing}, opacity ${duration} ${easing};
            transform: translateX(0) scale(1);
            opacity: 1;
          }
          .sdl-motion-chip[data-on='true'] {
            transform: translateX(360px) scale(1.06);
            opacity: 0.85;
          }
          @media (prefers-reduced-motion: reduce) {
            .sdl-motion-chip { transition-duration: 0ms; }
          }
        `}</style>
        <Box
          className="sdl-motion-chip"
          data-on={on ? 'true' : 'false'}
          position="absolute"
          top="50%"
          left="16px"
          mt="-18px"
          px={4}
          py={2}
          borderRadius="full"
          bg="action.primary"
          color="text.onGreen"
          fontSize="sm"
          fontWeight={600}
          style={{ willChange: 'transform, opacity' }}
        >
          Slashie
        </Box>
      </Box>
      <HStack gap={3}>
        <Box
          as="button"
          type="button"
          onClick={() => setOn((v) => !v)}
          px={4}
          py={2}
          minH="44px"
          borderRadius="lg"
          bg="action.primary"
          color="text.onGreen"
          fontSize="sm"
          fontWeight={600}
          cursor="pointer"
          style={{
            transition: `transform ${sdlMotion.duration.fast} ${sdlMotion.easing.standard}`,
          }}
          _hover={{ bg: 'action.primaryHover' }}
          _active={{ bg: 'action.primaryPressed', transform: 'scale(0.98)' }}
        >
          {on ? 'Reset' : 'Animate'}
        </Box>
        <Text fontFamily="mono" fontSize="xs" color="text.muted">
          transform + opacity · {duration} · {easing}
        </Text>
      </HStack>
    </Stack>
  )
}

### Reference usage

Apply motion in component styles by reading from `sdlMotion`, never by hardcoding `ms` or
cubic-bezier literals:

```tsx
import { sdlMotion } from '@/theme/styles'

const styles = {
  transitionProperty: 'transform, opacity',
  transitionDuration: sdlMotion.duration.moderate,
  transitionTimingFunction: sdlMotion.easing.standard,
}
```
